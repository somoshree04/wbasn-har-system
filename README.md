Wireless Body Area Sensor Network Based Human Activity Recognition System

Project Description

This project aims to design and implement a Wireless Body Area Sensor Network (WBASN) for monitoring physiological and motion-related signals from the human body. The collected data will be used to recognize and classify different human activities using sensor data.The system integrates multiple biomedical and motion sensors connected to a microcontroller. The sensor data will later be transmitted to a processing unit for analysis and activity recognition.


Sensors Used

The following sensors are planned to be integrated in the system:

MPU9250 – 9-axis motion sensor (accelerometer, gyroscope, magnetometer)

MAX30102 – Pulse oximeter for heart rate and SpO₂ monitoring

MLX90614 – Infrared temperature sensor for body temperature measurement

AD8232 – ECG sensor module for monitoring heart electrical activity


Hardware Platform

Arduino Uno – Used for initial sensor interfacing and testing

Raspberry Pi – Planned for data aggregation and processing
